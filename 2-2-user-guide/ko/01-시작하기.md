# 시작하기

## 목표
설치 → 첫 번째 채팅까지 최대한 빠르게.

## 가장 빠른 방법
1. **빠른 테스트**: `openclaw dashboard` → 브라우저에서 `http://127.0.0.1:18789/` 접속
2. **전체 설정**: `openclaw onboard --install-daemon` (마법사가 전부 처리)

## 사전 요구사항
- Node.js ≥ 22
- Brave Search API 키 (권장): `openclaw configure --section web`
- Windows: **WSL2 사용** (네이티브 Windows는 테스트 안 됨)

## 설치

```bash
# macOS/Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows PowerShell
iwr -useb https://openclaw.ai/install.ps1 | iex

# npm으로 설치
npm install -g openclaw@latest
```

## 온보딩 마법사

```bash
openclaw onboard --install-daemon
```

설정 항목:
- 로컬 vs 원격 게이트웨이
- 인증 (OAuth 또는 API 키)
- 채널 (WhatsApp, Telegram, Discord 등)
- 백그라운드 서비스 (launchd/systemd)
- 게이트웨이 토큰

## 게이트웨이 시작

```bash
# 상태 확인
openclaw gateway status

# 수동 실행 (포그라운드)
openclaw gateway --port 18789 --verbose
```

⚠️ **Bun 주의**: WhatsApp과 Telegram은 Bun에서 문제가 있음. **Node** 사용 권장.

## 빠른 확인

```bash
openclaw status
openclaw health
openclaw security audit --deep
```

## 채팅 연결

### WhatsApp
```bash
openclaw channels login
# WhatsApp → 설정 → 연결된 기기에서 QR 스캔
```

### Telegram / Discord
- 마법사가 토큰 설정 가능
- 또는 수동 설정 (채널 문서 참조)

## DM 보안 (페어링)
기본값: 모르는 발신자는 페어링 코드를 받고, 승인 전까지 처리되지 않음.

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code>
```

## 테스트 전송

```bash
openclaw message send --target +15555550123 --message "Hello from OpenClaw"
```

## 주요 경로
| 항목 | 위치 |
|------|------|
| 설정 | `~/.openclaw/openclaw.json` |
| 작업공간 | `~/.openclaw/workspace` |
| 자격증명 | `~/.openclaw/credentials/` |
| 세션 | `~/.openclaw/agents/<agentId>/sessions/` |
| 로그 | `/tmp/openclaw/` |
